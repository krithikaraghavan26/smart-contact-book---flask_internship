# Mini Project 1 – Day 6 (SQLite Integration)

This project demonstrates integrating SQLite with a Flask web application.

## Features
- Add contact details (name, phone, email)
- Store data persistently using SQLite
- View stored contacts
- Flash messages and basic error handling

## Technologies Used
- Python
- Flask
- SQLite (sqlite3)
- HTML, CSS

## How to Run
1. Create a virtual environment
2. Install dependencies
3. Run `python app.py`

> Note: The SQLite database file (`contacts.db`) is created automatically when the app runs.
